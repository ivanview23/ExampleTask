*Практическая часть курса по Java в Aston.*

Part1 - Реализация аналога класса MyHashSet с методами add() и remove() и Реализация аналога ArrayList с методами add(), get(), remove(), addAll()

Part2 - Создан класс Student, обязательное поле класса Student - List<Book> (минимум по 5 книг у каждого). Заполнить коллекцию студентами
        При помощи одного (не допускается объявления никаких промежуточных переменных, совсем никаких) стрима.
        
Part3 - Создание программы, которая будет записывать данные в файл и вычитывать данные из него. 
        Создание собственного исключения для ошибок при чтении/записи из файла.
        
Part4 - Реализация паттернов: Стратегия, Цепочка обязанностей, Билдер, Прокси, Декоратор, Адаптер.

Part5 - Реализация DeadLock и LiveLock. Создание двух потоков (поток 1 постоянно выводит "1", а поток 2 выводит "2"), которые будут бесконечно выводить в консоль "1" и "2" по очереди(начиная с "1")
